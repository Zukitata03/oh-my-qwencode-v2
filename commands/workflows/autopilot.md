Activate the **Autopilot** workflow from oh-my-qwencode.

Autopilot runs a full pipeline from idea to code: plan → implement → verify → QA.

## Pipeline
1. **Intent** — understand what the user wants.
2. **Plan** — create implementation plan.
3. **Implement** — execute the plan, delegating in parallel where possible.
4. **Verify** — run tests, builds, lint; confirm all pass.
5. **QA** — comprehensive quality check.

## Principles
- Proceed autonomously through the full pipeline.
- Ask only when a choice materially changes scope or is destructive.
- Deliver the full implementation — no scope reduction, no partial completion.
- Default to concise, evidence-dense progress reporting.
