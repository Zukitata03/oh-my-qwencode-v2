Activate the **UltraQA** workflow from oh-my-qwencode.

Cycling QA workflow: test → verify → fix → repeat until quality goal is met.

## Loop
1. **Test** — run full test suite or targeted tests.
2. **Verify** — read output, identify failures.
3. **Fix** — address each failure.
4. **Repeat** — re-run tests until all pass.

## Stop Conditions
- All tests pass green.
- User says stop.
- Fundamental blocker requires user input.
