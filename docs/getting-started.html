Getting Started

End-to-end setup for first-time users.

Prerequisites

  • Node.js 20+ and npm.
  • Qwen CLI installed and authenticated.
  • Git (recommended) for project workflows.

Installation

npm install -g @openai/qwen oh-my-qwencode
omq setup
omq doctor

Recommended Launch

omq --xhigh --madmax   # trusted environments
omq                    # standard launch

First Run Walkthrough

  1. Create or enter a project directory.
  2. Run omq setup to install prompts, skills, AGENTS.md, and configuration.
  3. Start Qwen Code with omq.
  4. Try an agent prompt: /prompts:architect "analyze auth module".
  5. Try a workflow skill: $team 3:executor "fix all TypeScript errors".

Prompt Instruction Source

OMQ layers project instructions by default (extends Qwen Code behavior, does not replace/bypass Qwen Code core system policies):

-c model_instructions_file="<cwd>/AGENTS.md"

Override behavior when needed:

OMQ_BYPASS_DEFAULT_SYSTEM_PROMPT=0 omq
OMQ_MODEL_INSTRUCTIONS_FILE=/path/to/instructions.md omq

Troubleshooting

IssueWhat to check
omq command not foundRe-run global install and ensure npm global bin is in PATH.
Prompts not availableConfirm files exist in ~/.qwen/prompts/ and re-run omq setup.
Skills not loadingVerify ~/.qwen/skills/*/SKILL.md files were installed.
Doctor reports config issuesRun omq doctor and apply the suggested fixes.
